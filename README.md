Descripción del proyecto: Aplicación de consola en Java basada en Pokémon. Permite iniciar o continuar una partida, capturar Pokémon, administrar el equipo, enfrentar gimnasios y desafiar al Alto Mando. La información del jugador y los Pokémon se almacena mediante archivos de texto.

Estructura:
JUEGO.java = Clase principal
JUGADOR.java, POKEMON.java, GIMNASIO.java, AltoMando.java = Clases del dominio
FUNCIONES.java y TablaTipos.java = Clases de apoyo y lógica

Archivos de texto:
Registros.txt
Pokedex.txt
Gimnasios.txt
Habitats.txt
Alto Mando.txt

Instrucciones de ejecución:

Abrir el proyecto en un compilador o IDE compatible con Java.

Verificar que todos los archivos .txt se encuentren dentro de la carpeta del proyecto.

Ejecutar la clase principal JUEGO.java.

Seguir las instrucciones mostradas en consola para jugar.
